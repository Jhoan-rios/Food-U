# 🍔 FoodU

## 📌 Introducción

FoodU es un proyecto desarrollado en Python que busca dar solución a una problemática real dentro de la Universidad de Medellín: la dificultad que enfrentan los estudiantes al momento de acceder a los servicios de alimentación en el campus.

Actualmente, los puntos de venta presentan altos niveles de congestión en horarios pico, lo que genera largas filas y tiempos de espera que afectan el desarrollo académico de los estudiantes. Esta situación se agrava debido a los cortos intervalos entre clases, limitando la posibilidad de adquirir alimentos de manera eficiente.

Además, no existe una plataforma centralizada que permita consultar información clave como la disponibilidad de productos, tiempos de preparación o el nivel de ocupación de cada punto de venta, generando desorganización, pérdida de tiempo y una experiencia poco óptima para los usuarios.

En respuesta a esta necesidad, FoodU propone un sistema de gestión de pedidos que permite a los estudiantes realizar pedidos anticipados, reducir tiempos de espera y optimizar su experiencia dentro del campus. A su vez, brinda a los vendedores herramientas para gestionar productos, pedidos y mejorar su servicio.

Este proyecto se desarrolla bajo el paradigma de Programación Orientada a Objetos (POO), permitiendo modelar de forma estructurada los diferentes componentes del sistema, como usuarios, productos, pedidos y vendedores.

---

## 🚀 Características

- 🛒 Pedidos anticipados  
- ⏱️ Sistema de turnos virtuales  
- 📊 Indicador de congestión  
- 🧠 Modo “entre clases”
- 🤖 Recomendaciones de menús con IA basadas en preferencias e historial de pedidos 
- 📦 Gestión de productos e inventario  
- ⭐ Sistema de calificaciones  
- 📜 Historial de pedidos  

---

## 🧠 Tecnologías utilizadas

- Python 🐍  
- Programación Orientada a Objetos (POO)  
- Consola / Interfaz básica  

---

## 🏗️ Estructura del proyecto

El sistema está basado en clases principales como:

- `Usuario`
- `Vendedor`
- `Producto`
- `Pedido`
- `SistemaFoodU`

Aplicando conceptos como encapsulación, herencia y polimorfismo.

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/foodu.git
```
2. Entrar a la carpeta:
```bash
cd foodu
```
3. Ejecutar el proyecto:
```bash
python main.py
```
